# 3DinJS
Учебный проект по трехмерной графике в JS.

## Папка FinalProject - итоговый проект.
### Описание проекта
Название: Полка Любимых Игр
Веб-приложение для хранения списка видеоигр (CRUD операции).

Из возможностей:
1. Добавление игры: название, жанр, статус, оценка. Также можно загрузить обложку, но только через URL
2. Редактирование игры
3. Удаление игры

Есть адаптивная верстка через Flexbox

#### Технологии
- **Бэкэнд:** Go + net/http (Стандартная библиотека)
- **БД:** PostgreSQL
- **Фронтэнд:** HTML, CSS, JS
- **Docker**

### Как запустить
